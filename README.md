# 同花顺7*24快讯抓取工具

## 功能说明

自动抓取同花顺财经网站（10jqka.com.cn）的7*24快讯板块数据，并导出到Excel文件。

### 核心特性

- **定时抓取**：每5分钟自动获取最新快讯数据
- **当天数据**：只抓取并保存当天的新闻数据
- **内容详情**：完整获取每条新闻的全部内容，不做截取
- **业绩筛选**：自动筛选业绩公告，保存为独立Excel文件
- **业绩股票筛选**：自动筛选价格在3-50元的业绩股票，保存到独立sheet
- **按天导出**：每天生成一个独立的Excel文件
- **数据去重**：避免重复保存相同新闻
- **持续运行**：程序持续运行，自动累积当天数据
- **网络检测**：自动检测网络连接状态，网络异常时跳过抓取
- **政策利好分析**：根据政策文本自动筛选相关受益股票，支持地区+行业双重筛选
- **股票代码管理**：自动保存全部A股数据到gp_codes.py文件，支持手动更新
- **基本面数据获取**：获取沪市主板、深市主板、创业板股票的基本面数据，包含行情和财务指标

## 使用方法

### 启动程序

**默认模式**：获取当天数据，每5分钟定时抓取

```bash
python news_scraper.py
```

**指定日期模式**：获取指定日期的数据

```bash
python news_scraper.py 2026-04-17
```

**政策利好股票分析模式**：根据政策文本自动筛选相关受益股票

```bash
python news_scraper.py --policy "广东省支持新能源汽车产业发展政策"
python news_scraper.py --policy "黑龙江省支持智能机器人产业发展若干政策措施"
python news_scraper.py --policy "福州市、徐州市、龙井市、宿州市等多地先后发放新一轮消费券"
```

**新闻分析模式**：根据新闻标题和内容自动识别利好行业/板块，并找出相关股票

```bash
python news_scraper.py --analyze "新能源汽车销量大增" "据最新数据显示，10月份新能源汽车销量同比增长80%，比亚迪、宁德时代等企业表现突出。"
python news_scraper.py --analyze "光伏产业链价格上涨" "受原材料价格上涨影响，光伏产业链各环节企业利润空间扩大。"
```

**基本面数据获取模式**：获取三大板块股票的基本面数据

```bash
python fetch_fundamental_data.py
```

## Excel数据格式

### 新闻数据Excel（news_YYYY-MM-DD.xlsx）

每个Excel文件包含以下列：

| 列名 | 说明 | 示例 |
|------|------|------|
| 时间 | 新闻发布时间 | 2026-04-18 16:42:46 |
| 标题 | 新闻标题 | 网传"环保新规推高养鸡成本致蛋价上涨"系谣言 |
| 内容详情 | 新闻完整内容 | 俄罗斯紧急情况部门通报，当地时间4月18日... |

### 业绩公告Excel（业绩公告_YYYY-MM-DD.xlsx）

包含3个sheet：

**Sheet 1: 业绩公告**
- 包含所有业绩公告信息（排除政策利好）
- 筛选条件：标题或内容包含"公告"或"营收"
- 列：时间、标题、内容详情

**Sheet 2: 政策利好**
- 包含政策相关利好信息
- 来源：政策关键词匹配 + 不包含"公告/营收"的业绩新闻
- 列：时间、标题、内容详情

**Sheet 3: 业绩股票(3-50元)**
- 从新闻标题中提取股票名称（"："之前的文字）
- 自动匹配股票代码和当天价格
- 筛选条件：
  - 价格在3-50元之间
  - 排除ST股票（包含*ST）
  - 排除科创板股票（688开头）
- 列：代码、名称、价格、涨跌幅、新闻标题、新闻时间
- 排序：按价格从高到低

**示例输出：**
```
已保存 110 条业绩公告到 news_data\业绩公告_2026-04-17.xlsx（91 条股票公告 + 19 条政策利好）
已筛选 48 条业绩股票（价格3-50元）
```

包含3个sheet：

- 沪市主板股票（600/601/603/605开头）

**Sheet 2: 深市主板**

**Sheet 3: 创业板**

## 数据说明

- **数据来源**：同花顺财经网站7*24快讯板块、腾讯行情API、同花顺财务数据API
- **更新频率**：每5分钟自动更新（新闻数据）
- **保存周期**：只保存当天的数据
- **文件命名**：`news_YYYY-MM-DD.xlsx`
- **数据存储**：`news_data` 目录
- **数据累积**：程序持续运行时，会自动累积当天的新数据，每次只保存新增部分

## 注意事项

1. 程序需要保持运行状态才能持续抓取数据
2. 确保网络连接正常
3. 数据文件会按天自动保存，每天一个文件
4. 内容摘要会自动提取，标题保持原文不变
5. 程序会自动去重，不会重复保存相同新闻
6. 政策利好股票分析模式会自动剔除价格<3和价格>50的股票、科创板股票（688开头）和ST股票
7. 政策识别支持国家政策和地方政策，地方政策可结合地区+行业双重筛选
8. 行业关键词映射基于同花顺官方行业分类代码
9. 基本面数据获取使用多线程并发，提高获取速度
10. 财务数据来源于同花顺财务摘要接口，部分股票可能无最新财务数据

## 技术实现

### 数据抓取

- 使用同花顺API接口获取快讯数据
- 支持分页获取历史数据
- 自动解析JSON格式响应

### 数据处理

- 时间戳转换为可读时间格式
- 自动提取内容摘要（移除HTML标签，截取150字）
- 按日期分类存储

### 定时任务

- 使用schedule库实现定时调度
- 每5分钟执行一次抓取任务
- 支持程序持续运行

### 基本面数据获取

- 使用腾讯行情API获取实时行情数据
- 使用akshare库获取同花顺财务数据
- 多线程并发获取财务数据，提高速度
- 自动过滤ST股票
